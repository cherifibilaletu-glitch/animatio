# معرض تصاميم القطاعات (Sectors Showcase)

موقع ثابت (HTML/CSS/JS فقط — بدون خطوة بناء) يعرض **٩ طرق أنيقة ومتحركة** لعرض القطاعات، لتختار الأنسب لموقعك.

## التصاميم التسعة
1. شبكة بِنتو (Bento Grid)
2. خلية النحل السداسية (Hexagon)
3. العرض المنقسم (Split Showcase)
4. شريط البطاقات (Carousel)
5. المدار المحسّن (Orbit)
6. فسيفساء الصور (Mosaic)
7. البطاقات القالبة (Flip Cards)
8. بطاقات ثلاثية الأبعاد (3D Tilt)
9. الشريط المتحرك (Marquee)

## التشغيل على GitHub Pages
1. افتح **Settings ← Pages**.
2. في **Source** اختر **Deploy from a branch**.
3. اختر الفرع **main** والمجلّد **/ (root)** ثم **Save**.
4. انتظر دقيقة ثم افتح:

```
https://cherifibilaletu-glitch.github.io/animatio/
```

> لا يحتاج أي workflow أو خطوة بناء.
